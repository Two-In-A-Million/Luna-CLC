# Luna-skill-calculator
This is a luna skill calculator. This project was made to be embbeded to antoher web (widget style!)


To run this project:
- cd to backend and run "node app.js" > this will run the backend server locally on port 3000
- open new terminal, cd to frontend, and run "npm run dev -- --host" > this will run the frontend server locally on port 5173
- open another terminal, cd to Postgres, run "docker compose up -d" (run it on wsl if needed) > this will run the postgres db on your local machine
